## Controls

##### Mouse and Keyboard

WASD / Arrows = Movement

Space = Attack/Select Menu

E = Interact/Talk/Select Level Up

R = Roll/Close Menu

Y = Drink Potion

Start/Middle = Pause

##### XBOX 360 Controller

Arrow/Left Stick = Movement

A = Attack/Select Menu

X = Interact/Talk/Select Level Up

B = Roll/Close Menu

Y = Drink Potion

Start/Middle = Pause
